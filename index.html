Why html default style is ugly?

Who knows, maybe performance, maybe history, but i knows one thing for sure, using class="alert alert-primary" can't be the solution. Why am i using a ton of classes and ids to style simple components?

Lot's of questions around web developement these days, but answers started rising on the horizon.

My goal is to show my solution to many many problems. First, i'll point out some of these problems, than i'll give you a possible solution (at ctrl+shift+i > Elements > Head > <link rel="stylesheet" href="base.css">, and this whole document is just a tech demo about how powerfull can it be).

No more jquery?

The new version of bootstrap stops using jquery and this is the good direction. Jquery is unneccessary and we really don't need it.

Classes classes classes...

In theory we have components for the most usecases, and div, span for special cases. In real life we have divs, spans, and classes, and sometimes sections, heads.

These days we have many many tags with semantic meaning, but special behavior is missing. The layout spice still missing from these components, and this is the reason we use

<div class="col-4">

Div forest

My other pain is the huge markup for every single component. when you see 10 divs, and the result is a simple layout with a coumple of components.

Solution?

In my opinion Bootstrap can't be it. More like an "extension" for the default style. Like base.css, this is actually an experiment about it. A couple hundered lines of css addition to the default beaviors, based on tagnames and relations between them.

You can see, i'm that motherfucker person, but this is the way to make fast, simple, and not ugly websites. Html is rich. We can use it to make our website more colorfull with minimal styling. Better to have a nice colorfull website, but also better to keep it simple as possible.

You should use classes but frameworks should not. You have to build your layout with Grid/Flexbox, but browsers should give you some default options, only with html tags for simpler designs.
